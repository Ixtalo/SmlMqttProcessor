example-data-sender

Simple Python script to simulate libsml’s sml_server textual, decoded output. It can be used as input to smltextmqttprocessor.py.

Usage

python3 scripts/example-data-sender/example_data_sender.py | python3 smltextmqttprocessor.py --no-mqtt --debug config.local.ini -
